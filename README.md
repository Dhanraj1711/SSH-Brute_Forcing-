# SSH-Brute_Forcing-
IMPORTANT — READ BEFORE USING
This project is intended only for lawful, authorized security testing, training, and research in controlled environments (your own machines, a stolen-credentials lab, or a test network where you have explicit written permission). Do not use this code against systems you do not own or do not have explicit permission to test. Misuse may be illegal and harmful.

Python 3.8+ (or list your runtime)
Virtual environment recommended 
Do not change the target to systems you do not own/have permission for.
Create a dedicated VM or container that you own.
Install OpenSSH server on that VM.
Run the simulator against the VM’s private IP only.
